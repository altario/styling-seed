#Buttons

Buttons

Button Types & Sizes

Define a button size with btn-tiny, btn-small, btn, btn-big or btn-huge. The default button size is btn.

Button Normal

<button type="button" class="btn btn-primary btn-tiny">Button Tiny</button>

Button Outline

<button type="button" class="btn btn-outline-primary btn-tiny">Button Tiny</button>

Button Ghost

<button type="button" class="btn btn-ghost-primary btn-tiny">Button Tiny</button>

Buttons with Icons

Buttons with icons are an extend of a basic button but with an additional class. For example, btn-icon-left will let the element with text know that is has to give space for an icon. The icon has to be placed straight after the button and should be a Font Awesome icon.
