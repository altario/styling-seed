Buttons Buttons

Types and Sizes

Define a button size with btn-tiny, btn-small, btn, btn-big or btn-huge. The default button size is btn.

Button Normal

<button type="button" class="btn btn-primary btn-tiny">Button Tiny</button>

Button Outline

<button type="button" class="btn btn-outline-primary btn-tiny">Button Tiny</button>

Button Ghost

<button type="button" class="btn btn-ghost-primary btn-tiny">Button Tiny</button>

With Icons (left)

Buttons with icons are an extend of a basic button but with an additional class. For example, btn-icon-left-calendar will let the element with text know the side of the icon and that it has to give space for it. The icon falls back to the Font Awesome icons but if icons are set on the $iconfont map in the _iconfont.scss, FontAwesome will be overrided.

Button Normal

<button type="button" class="btn btn-primary btn-small btn-icon-left btn-icon-calendar">Button Small</button>

Button Ghost

Loading

Buttons with btn-is-loading class override icon classes such as the example with btn-icon-left-calendar or btn-icon-right-calendar.

Button Normal

<button type="button" class="btn btn-primary btn-icon-left-calendar btn-is-loading" disabled>Button Small</button>

Disabled

Disabled buttons receive an disabled attribute.

Button Normal

<button type="button" class="btn btn-primary btn-icon-calendar" disabled>Button Small</button>

Button Ghost
